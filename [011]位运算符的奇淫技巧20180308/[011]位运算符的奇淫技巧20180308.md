7. 无符号右移>>>

各种位运算符的妙用：
- 利用&判断奇数偶数
    +  n & 1 : 结果为1则n为奇数，结果为0则n为偶数
- 使用~, >>, <<, >>>, |来取整(无符号右移>>>不能对负数取整)
    + `~~6.123 = 6`
    + `~~-6.123 = -6`
    + `6.123 >> 0 = 6`
    + `-6.123 >> 0 = -6`
    + `6.133 << 0 = 6`
    + `-6.123 << 0 = 6`
    + `6.123 >>> 0 = 6;`
- 使用^来完成值交换
    ```javascript
    var a = 3,b = 4;
    a ^= b;
    b ^= a;
    a ^= b;
    ```
- 使用<<来实现2的n次方
    + 1 << n (表示2的n次方)
